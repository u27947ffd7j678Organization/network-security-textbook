# 用語集

> 本教科書で使用する用語の正式な定義をまとめます。
> 初出時は各章で簡潔に説明し、必要に応じて本ファイルを参照します。

| 用語 | 英語 | 概要 | 初出 |
|---|---|---|---|
| ネットワーク | Network | 複数の機器が共通のルールで通信する仕組み | 第1部 第1章 |
| プロトコル | Protocol | 機器同士が通信するための共通ルール | 第1部 第1章 |
| LAN | Local Area Network | 家庭や組織など限定された範囲のネットワーク | 第1部 第1章 |
| WAN | Wide Area Network | 離れたLAN同士を接続する広域ネットワーク | 第1部 第1章 |
| インターネット | Internet | 世界中のネットワークが相互接続された集合体 | 第1部 第1章 |
| Web | World Wide Web | HTTP/HTTPSを利用して情報を提供するサービス | 第1部 第1章 |
| クライアント | Client | サービスを利用する側 | 第1部 第1章 |
| サーバ | Server | サービスを提供する側 | 第1部 第1章 |
| DNS | Domain Name System | ホスト名とIPアドレスなどを対応付ける仕組み | 第1部 第1章 |
| IPアドレス | IP Address | ネットワーク上の機器やインターフェースを識別する論理アドレス | 第1部 第1章 |
| TCP | Transmission Control Protocol | 順序制御や再送により信頼性のある通信を提供するプロトコル | 第1部 第1章 |
| UDP | User Datagram Protocol | 再送や接続管理を最小限にした軽量な通信プロトコル | 第1部 第1章 |
| HTTP | Hypertext Transfer Protocol | Webにおける要求と応答を定めるプロトコル | 第1部 第1章 |
| HTTPS | Hypertext Transfer Protocol Secure | TLSで保護されたHTTP通信 | 第1部 第1章 |
| TLS | Transport Layer Security | 通信相手の確認と通信の暗号化を行うプロトコル | 第1部 第1章 |
| パケット | Packet | ネットワークで転送されるデータの単位 | 第1部 第2章 |
| ヘッダ | Header | 宛先や順序など通信制御に使う付加情報 | 第1部 第2章 |
| ルータ | Router | 異なるIPネットワーク間でパケットを転送する機器 | 第1部 第2章 |
| OSI参照モデル | OSI Reference Model | 通信機能を七つの層に整理した概念モデル | 第1部 第3章 |
| カプセル化 | Encapsulation | データへ各層の制御情報を順番に付加する処理 | 第1部 第3章 |
| フレーム | Frame | データリンク層で扱うデータ単位 | 第1部 第3章 |
| セグメント | Segment | TCPが扱うデータ単位 | 第1部 第3章 |
| TCP/IP | TCP/IP | インターネット通信を支えるプロトコル群の総称 | 第1部 第4章 |
| ポート番号 | Port Number | 端末上のアプリケーションやサービスを識別する番号 | 第1部 第4章 |
| ネットワークスタック | Network Stack | OSが提供する階層化された通信処理の集合 | 第1部 第4章 |
| ソケット | Socket | アプリケーションが通信機能を利用するための窓口 | 第1部 第4章 |
| NIC | Network Interface Card | コンピュータをネットワークへ接続するインターフェース | 第1部 第5章 |
| MACアドレス | MAC Address | データリンク層でインターフェースを識別するアドレス | 第1部 第5章 |
| スイッチ | Switch | 主に同一LAN内でフレームを転送する機器 | 第1部 第5章 |
| アクセスポイント | Access Point | 無線LAN端末を有線LANなどへ接続する機器 | 第1部 第5章 |
| Firewall | Firewall | 条件に基づいて通信を許可または遮断する仕組み | 第1部 第5章 |
| デフォルトゲートウェイ | Default Gateway | 別ネットワーク宛て通信を最初に渡すルータ | 第1部 第5章 |
| ルーティングテーブル | Routing Table | 宛先に応じた転送先を記録した表 | 第1部 第5章 |
| ペイロード | Payload | ヘッダを除いた、実際に運ぶ内容 | 第1部 第6章 |
| MTU | Maximum Transmission Unit | 一つのリンクで運べるIPパケットサイズの上限 | 第1部 第6章 |
| フラグメンテーション | Fragmentation | 大きなIPパケットを分割する処理 | 第1部 第6章 |
| TTL | Time To Live | IPパケットが通過できるルータ数を制限する値 | 第1部 第6章 |
| チェックサム | Checksum | 転送中のデータ破損を検出するための値 | 第1部 第6章 |
| カーネル | Kernel | ハードウェアとシステム資源を管理するOSの中核 | 第1部 第7章 |
| ネットワークインターフェース | Network Interface | Linuxが通信の出入口として扱う単位 | 第1部 第7章 |
| ループバックインターフェース | Loopback Interface | 端末自身との通信に使用する仮想インターフェース | 第1部 第7章 |
| IPv4 | Internet Protocol version 4 | 32ビットのIPアドレスを使うIP | 第2部 第1章 |
| IPv6 | Internet Protocol version 6 | 128ビットのIPアドレスを使うIP | 第2部 第1章 |
| グローバルIPアドレス | Global IP Address | インターネットの経路制御に使われる一意なIPアドレス | 第2部 第1章 |
| プライベートIPアドレス | Private IP Address | 組織内や家庭内で再利用できるIPv4アドレス | 第2部 第1章 |
| ループバックアドレス | Loopback Address | 端末自身との通信に使用するIPアドレス | 第2部 第1章 |
| サブネット | Subnet | IPネットワークを分割したネットワーク範囲 | 第2部 第2章 |
| サブネットマスク | Subnet Mask | IPv4アドレスのネットワーク部分を示す値 | 第2部 第2章 |
| プレフィックス長 | Prefix Length | IPアドレスの先頭からネットワーク部分となるビット数 | 第2部 第2章 |
| CIDR | Classless Inter-Domain Routing | アドレスとプレフィックス長でネットワーク範囲を表す方式 | 第2部 第2章 |
| ネットワークアドレス | Network Address | サブネット自体を表す、ホスト部がすべて0のIPv4アドレス | 第2部 第2章 |
| ブロードキャストアドレス | Broadcast Address | IPv4サブネット内の全端末宛てに使うアドレス | 第2部 第2章 |
| ユニキャスト | Unicast | 一つの宛先へ送る通信方式 | 第2部 第3章 |
| ブロードキャスト | Broadcast | 同じ範囲内の全端末へ送る通信方式 | 第2部 第3章 |
| マルチキャスト | Multicast | 特定のグループへ送る通信方式 | 第2部 第3章 |
| ARP | Address Resolution Protocol | IPv4アドレスに対応する同一リンク内のMACアドレスを調べるプロトコル | 第2部 第4章 |
| ARPキャッシュ | ARP Cache | IPv4アドレスとMACアドレスの対応を一時保存した情報 | 第2部 第4章 |
| 近隣探索 | Neighbor Discovery | IPv6で近隣情報などを解決するICMPv6の仕組み | 第2部 第4章 |
| ルーティング | Routing | 宛先IPアドレスに応じて次の転送先を選ぶ処理 | 第2部 第5章 |
| デフォルトルート | Default Route | 個別経路に一致しない宛先へ使う経路 | 第2部 第5章 |
| 最長プレフィックス一致 | Longest Prefix Match | 一致する経路のうち最も具体的なプレフィックスを選ぶ規則 | 第2部 第5章 |
| 静的ルーティング | Static Routing | 管理者が経路を設定する方式 | 第2部 第5章 |
| 動的ルーティング | Dynamic Routing | ルータ同士がプロトコルで経路情報を交換する方式 | 第2部 第5章 |
| ICMP | Internet Control Message Protocol | IP通信の状態やエラーを通知する制御プロトコル | 第2部 第6章 |
| ping | ping | ICMP Echoを利用して到達性や往復時間を確認するコマンド | 第2部 第6章 |
| traceroute | traceroute | TTL超過応答を利用して経路を推測するコマンド | 第2部 第6章 |
| DHCP | Dynamic Host Configuration Protocol | 端末へIPアドレスなどの設定を自動配布するプロトコル | 第2部 第7章 |
| リース | Lease | DHCPがネットワーク設定を貸し出す有効期間 | 第2部 第7章 |
| DHCPリレー | DHCP Relay | 異なるサブネットのDHCPクライアントとサーバを中継する機能 | 第2部 第7章 |
| NAT | Network Address Translation | パケットのIPアドレスを変換する技術 | 第2部 第8章 |
| NAPT | Network Address Port Translation | IPアドレスとポート番号を変換する技術 | 第2部 第8章 |
| 送信元NAT | Source NAT | パケットの送信元IPアドレスを変換するNAT | 第2部 第8章 |
| 宛先NAT | Destination NAT | パケットの宛先IPアドレスを変換するNAT | 第2部 第8章 |
| ポートフォワーディング | Port Forwarding | 特定ポート宛ての通信を内部の宛先へ転送する設定 | 第2部 第8章 |
| eBPF | extended Berkeley Packet Filter | Linuxカーネル内で安全にプログラムを実行する仕組み | 第8部 |
