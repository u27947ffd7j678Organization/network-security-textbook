# ネットワーク・セキュリティ教科書

ネットワークとセキュリティを、基礎から設計・運用・実践まで体系的に学ぶための教科書です。本文はUTF-8のMarkdownで管理し、将来Pandoc等を利用してHTML/PDFへ変換できる構成にしています。

## 執筆方針

- 読者が再現できる説明を優先し、前提条件・手順・期待結果を明記します。
- 用語は初出時に定義し、表記は教科書全体で統一します。
- コマンドや設定例には、対象OS・バージョン・権限・安全上の注意を添えます。
- 図表には番号、題名、本文からの参照、出典または作成者を記載します。
- 実在する組織のアドレスや認証情報を例に使わず、予約済みの例示用アドレスを使用します。
- 章ごとに学習目標、前提知識、本文、演習、まとめ、参考資料を設けます。
- 変更はレビューを経て `main` ブランチへ取り込みます。

## HTML/PDFへの変換

[Pandoc](https://pandoc.org/)を導入後、章ファイルを目次順に指定して変換します。

```bash
pandoc --defaults=変換設定/html.yaml 本文/**/*.md
pandoc --defaults=変換設定/pdf.yaml 本文/**/*.md
```

シェルによっては `**` が展開されないため、その場合はファイルを明示的に列挙してください。PDF生成には別途、日本語対応のLaTeXエンジンとフォントが必要です。
